Infinite Unpossibility

Controls:

Before Game

Number Keys, Up, Down - Select menu options
Letter Keys, Backspace - Change name or epithet
Space or Enter - Perform selected action
-/+, Mouse Wheel - Change the parameter of the highlighted action
, - Randomize the deity options

In Game

, - Toggle history filter
. - Toggle influence vision
/ - Toggle map names

Number Keys - Select actions
Space or Enter - Perform selected action
-/+, Mouse Wheel - Change the parameter of the highlighted action

Mouse or Arrow Keys - Move the cursor
